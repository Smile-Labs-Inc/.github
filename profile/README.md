# Smile Labs

We're a student engineering team from Sri Lanka. Most of what's here started as a
hackathon or competition entry. A few grew past that.

**[smilelabs.online](https://smilelabs.online/)**

## Competition builds

| Project | What it is |
| --- | --- |
| [relic-ring-router](https://github.com/Smile-Labs-Inc/relic-ring-router) | Low-latency interplanetary packet-routing simulator. Built for Launch'26. |
| [sentilex-ai-advocate](https://github.com/Smile-Labs-Inc/sentilex-ai-advocate) | An autonomous AI legal advocate for trauma victims in Sri Lanka. |
| [Proxy-Maze](https://github.com/Smile-Labs-Inc/Proxy-Maze) | Real-time proxy-pool monitoring API. Built for ProxyMaze'26 (Torch Labs). |
| [Data-Storm-2026](https://github.com/Smile-Labs-Inc/Data-Storm-2026) | Our entry for the Data Storm 2026 data-science competition. |
| [fluxcard-firmware](https://github.com/Smile-Labs-Inc/fluxcard-firmware) | Firmware for FluxCard, a battery-less, NFC-powered point-of-care diagnostic. |
| [iwb25-192-sharks](https://github.com/Smile-Labs-Inc/iwb25-192-sharks) | Rewrites sensationalized news into neutral reporting with NLP. Built for Innovate with Ballerina 2025. |
| [SmartBabyLK-X2](https://github.com/Smile-Labs-Inc/SmartBabyLK-X2) | A digital health companion that turns paper infant-health records into AI-assisted care. |
| [growa-core-engine](https://github.com/Smile-Labs-Inc/growa-core-engine) | Crop guidance, market data, and pest detection for Sri Lankan smallholder farms. |
| [Persona](https://github.com/Smile-Labs-Inc/Persona) | AI personas for market research. |
| [AdaptIQ](https://github.com/Smile-Labs-Inc/AdaptIQ) | Adaptive, personalized learning. Early stage. |
